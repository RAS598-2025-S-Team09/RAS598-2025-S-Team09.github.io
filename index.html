This project involves making a UR5 robot (by Universal Robots) learn to play table tennis against a human or another UR5 robot and an autonomous Turtlebot that can drive autonomously to track and pick up fallen balls. The project helps us understand various tasks that can be acheived when one can integrate Vision, Motion and Controls.

Technology Used

The technical stack includes the following:

  • ROS 2 Humble: This middleware/framework is responsible for communication between various modules like perception, planning, controls involving sensors and actuators like camera, IMU, joint motors, etc.
  • Gazebo: This is the simulation environment used to work on various scenarios in the real world. This also captures various interactions between the different components like robot arm, ball, paddle and table
  • OpenCV: This library is used for Computer Vision and Image Processing that helps locate amd track objects in the surrounding leading to better interaction.
  • Reinforcement Learning: This technology helps the robot learn the art of taking appropriate action given a state or observation. Unlike traditional Computer Vision, where actions are hard-coded against actions, here the robot learns various actions and their use in various scenarios and can apply it in unknown scenarios.
  • ROS 2 Navigation: This is required for the Turtlebot to navigate in a real environment as it is tasked to pick up fallen tennis balls.
  • Sensor Fusion: The technique combines data from various sensors to give the robot a better estimate of the world which inturn helps it take better actions.
Sensors Used

The following sensor and actuators are used in this project:

  • RGBD Camera: This sensor gives information about visual aspects of an environment along with the tactile aspects like distance. The sensor uses IR sensors to gain depth information along with regular RGB cameras.
  • IMU: This sensor gives information about the various forces acting on an the object (Turtlebot or robot arm). It consists of Accelometer, Gyroscope and sometime a Magnetometer. One can gain velocity and displacement from this sensor too.
  • UR5 Robot: This is the fifth collaborative robot (CoBot) in the UR series of robots. The specifications of this robot can be seen here.
  • Turtlebot: This is a mobile robot with sensors like RGBD camera, IMU, LiDAR, etc that can be turned into an autonomous vehicle by integrating navigation, localization and mapping. The specifications can be viewed here
